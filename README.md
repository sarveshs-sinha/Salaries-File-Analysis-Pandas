# Salaries-File-Analysis-Pandas

# 📊 Salaries Dataset Analysis with Python & Pandas

A beginner-friendly Exploratory Data Analysis (EDA) project using **Python**, **Pandas**, and **NumPy**. This notebook explores the Salaries dataset by answering common data analysis questions and performing data cleaning.

## 📌 Project Overview

The goal of this project is to practice data analysis using Pandas by:

- Loading and exploring the dataset
- Cleaning missing and unnecessary data
- Finding useful insights from employee salary records
- Performing statistical analysis
- Using filtering, grouping, and aggregation

## 📈 Analysis Performed

The notebook includes the following tasks:

1. Display the first 10 rows
2. Display the last 10 rows
3. Find the dataset shape (rows and columns)
4. Display dataset information
5. Check missing values
6. Remove unnecessary columns
7. Generate descriptive statistics
8. Find the top 5 most common employee names
9. Count unique job titles
10. Find job titles containing "Captain"
11. Display employees from the Fire Department
12. Find minimum, maximum, and average Base Pay
13. Replace "Not Provided" values with `NaN`
14. Remove rows with excessive missing values
15. Find the job title of **ALBERT PARDINI**
16. Find ALBERT PARDINI's total salary including benefits
17. Find the employee with the highest Base Pay
18. Calculate average Base Pay by year
19. Calculate average Base Pay by job title
20. Find the average Base Pay of ACCOUNTANTS
21. Display the top 5 most common job titles
